Description:

The purpose of the project is to introduce you to (1) constructors/overloaded constructors, (2) destructors, and (3) pointers and dynamic memory. You will also be presented with a new parsing problem, and of course you will continue to sharpen your skills in object-oriented design and programming in C++.

In this program, you will design and implement a simple "virtual pet" game. Each "virtual pet" will have a name and a hunger level (at level 0 the pet "passes out" and at level 10 the pet is full and will not eat any more). The hunger level of each pet decreases by 1 level every 2 minutes.

At the start of the game the program will create 3 "virtual pets" for your player to interact with. Each pet will have a name (hard-coded) and a random initial hunger level, between 5-10. Your player will have the option to feed each pet individually (i.e., the player must select a specific pet to feed). When a pet "passes out," the player will be prompted to name a new pet, which will take the place of the pet that passes out, and be given a random initial hunger level, again between 5 and 10.

Some technical details:

  • design and implement a class to model the virtual pets, with the following:
    • a default constructor
    • at least one overloaded constructor
    • a destructor that displays a message when it is invoked, such as "Spot has passed out." (Spot being the name of the pet in this example); use the 'cerr' stream for this output
  • you may use the timer code provided by this class
  • design and implement a user interface class, with the following:
    • use pointers to allocate/release the three required pet objects
Objectives:
  • implement and use a default constructor
  • implement and use an overloaded constructor
  • implement a destructor and identify when it will be invoked
  • use pointers to allocate and access dynamic memory
  • correctly manage dynamically allocated memory
